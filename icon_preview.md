# Material Icons 图标预览指南

## 在线预览网站
1. **官方 Material Icons 网站**：https://fonts.google.com/icons
   - 可以搜索、预览所有图标
   - 支持按分类浏览
   - 显示图标名称，可直接在代码中使用

2. **Flutter Icons 预览**：https://materialdesignicons.com/
   - Material Design Icons 的完整列表

## 与IP/网络相关的图标推荐

### 当前使用的图标
- `Icons.dns` - DNS图标（当前使用）

### 其他可能合适的图标
- `Icons.language` - 语言/网络图标
- `Icons.network_check` - 网络检查（之前使用）
- `Icons.router` - 路由器图标
- `Icons.settings_ethernet` - 以太网设置
- `Icons.wifi` - WiFi图标
- `Icons.cloud` - 云/网络图标
- `Icons.devices` - 设备图标
- `Icons.computer` - 计算机图标
- `Icons.storage` - 存储/服务器图标
- `Icons.public` - 公共/互联网图标
- `Icons.cloud_queue` - 云队列图标
- `Icons.network_ping` - 网络ping图标（如果可用）
- `Icons.signal_wifi_4_bar` - WiFi信号图标
- `Icons.cable` - 网线图标

## 在代码中使用
```dart
Icon(Icons.icon_name)
```

## 快速测试方法
可以在 `main_screen.dart` 中临时修改图标名称，运行应用查看效果。
