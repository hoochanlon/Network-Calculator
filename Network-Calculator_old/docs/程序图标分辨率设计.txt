# 程序图标分辨率设计
'''
**图标分辨率选择**

**16x16** :

* 用于任务栏、文件资源管理器中的小图标
* 传统的浏览器标签页图标

**32x32** :

* 用于程序的任务栏图标和一些系统窗口
* 较高分辨率的favicon，适用于更现代的显示设备和浏览器

**48x48** :

* 适用于某些桌面快捷方式或小尺寸的快捷方式图标
* 可用于显示在某些浏览器和设备的地址栏图标

**64x64** :

* 用于高分辨率屏幕上的应用程序图标。
* 可用于显示在某些浏览器和设备的地址栏图标。

**128x128** :

* 更大尺寸的图标，适用于一些高分辨率显示器或在“添加到开始菜单”等位置显示。
* 推荐用于Apple设备的“添加到主屏幕”图标。

**256x256** :

* 用于Windows Vista及以上版本的“开始菜单”及系统图标，通常也用于高分辨率显示。

**512x512** :

*  用于支持高分辨率的设备（例如Android设备、Web应用、mac等）。
'''
