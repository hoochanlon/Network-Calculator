# 网络计算器 功能概览（无图版 / CN + EN + JP）
> 提供中文、英文、日文三语概要；无截图，适合终端/纯文本环境。

## 功能模块
- IP 地址 / 子网计算：输出网络地址、广播地址、可用 IP 等信息。  
- 子网掩码计算（桌面端示例）：按主机数或掩码计算可用网段。  
- 超网拆分：输入超网（CIDR）与目标掩码，生成拆分后的子网列表。  
- 路由汇总：支持多条 CIDR 聚合，提供不同算法以对比汇总结果。  
- 进制转换：支持 IP/数值在十进制、二进制、十六进制之间互转。  
- 历史记录：支持搜索、导出、导入；清除计算器状态会清空所有输入/结果缓存（Web 和桌面均生效）。  

## 平台特性
- Web 端：导出历史会触发浏览器下载 JSON；导入通过文件内容；清除状态在当前浏览器环境生效。  
- 桌面端：导出历史保存到系统下载目录（若不可用则退回应用文档目录）；导入使用文件路径；窗口尺寸和存储目录可定制。  

## 语言与主题
- 多语言：简体中文、繁体中文（中国香港）、English (US)、日本語。  
- 主题：浅色 / 深色 / 跟随系统；支持自定义颜色主题。

## English Summary (No Images)
- **Modules**: IP/Subnet calculation; subnet mask (desktop); supernet split; route aggregation (multiple algorithms); base conversion (dec/bin/hex); history (search/export/import, clear calculator states).  
- **Platforms**: Web exports trigger browser download; imports via file content; clear states in current browser context. Desktop exports to system Downloads (fallback to app docs); imports via file path; window size & storage path configurable.  
- **Languages & Themes**: Simplified Chinese, Traditional Chinese (Hong Kong), English (US), Japanese; light/dark/system themes; customizable color themes.

## 日本語サマリー（画像なし）
- **機能モジュール**：IP/サブネット計算、サブネットマスク計算（デスクトップ）、スーパーネット分割、ルート集約（複数アルゴリズム）、基数変換（10/2/16 進）、履歴（検索・エクスポート・インポート、計算機状態クリア）。  
- **プラットフォーム**：Web はエクスポート時にブラウザで JSON ダウンロードをトリガー、インポートはファイル内容を使用、状態クリアは現在のブラウザ環境で有効。デスクトップはシステムのダウンロードフォルダ（無ければアプリドキュメント）へ保存、インポートはファイルパス指定、ウィンドウサイズと保存先を設定可能。  
- **言語とテーマ**：簡体中文・繁體中文（香港）・English (US)・日本語。ライト/ダーク/システム連動テーマ、カスタム配色をサポート。
